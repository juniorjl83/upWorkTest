# UpWorkTest
Java command line program with following features

- Creates a number of elements
- Elements can be linked between them
- Searches if two elements are linked

## Running

### Running from IDE

 - Build the application through your IDE
 - Run UpWorkTestApplication.java

### Running from CLI

To run through the command line interface, first configure environment variables for all environment-specific properties, then:

```bash
mvn clean package
java -jar target/<generated-jar>
```
